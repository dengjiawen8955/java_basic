### 二、什么是静态语言什么是动态语言?

#### 1.动态语言JavaScript(可以在运行是改变自身的结构)

```javascript
function f(){
var x = "var a = 3;var b = 4;alert(a+b)";
eval(x);//可以改边x的值，String改为其他.
}
```

#### 2.静态语言比如Java，C, C++,是静态语言,但是Java通过反射机制可变伪动态语言.
