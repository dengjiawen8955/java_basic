# Java-GUI-GreadySnack
# Java GUI 贪吃蛇项目,带素材.
### 一、3个类，
#### Data.java 
Data.java 用于 初始化图片资源
#### StartGame.java
StartGame.java 用于初始化一个 window 窗口和 调用启动一个最重要的游戏面板 GamePanel.java
#### GamePanel.java
最重要的游戏面板，游戏的核心，
包括数据添加，时间监听，计时器都在里面。
